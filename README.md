# fabric-ultravox

Ultravox FE integration with its API

# Install the dependencies using below code

npm install

# Sign up on Ultravox site and generate API key: Link: https://app.ultravox.ai/

# Create .env.local file and copy the below code and replace <API-KEY> your API key generate in above step

VITE_ULTRAVOX_API_KEY=<API-KEY>

# Run the code

npm run dev
